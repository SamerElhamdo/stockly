# Stockly Mobile App

تطبيق الهاتف المحمول لنظام إدارة المخزون والفواتير Stockly، مبني باستخدام React Native و Expo.

## المميزات

- 🔐 تسجيل الدخول الآمن
- 📊 لوحة تحكم شاملة مع الإحصائيات
- 👥 إدارة العملاء
- 📦 إدارة المنتجات والفئات
- 🧾 إنشاء وإدارة الفواتير
- 📱 واجهة مستخدم متجاوبة ومتوافقة مع الأجهزة المحمولة
- 🌙 دعم الوضع المظلم
- 🔄 تحديث البيانات بالسحب للأسفل

## التقنيات المستخدمة

- **React Native** - إطار العمل الأساسي
- **Expo** - منصة التطوير والنشر
- **React Navigation** - التنقل بين الشاشات
- **React Native Paper** - مكتبة واجهة المستخدم
- **Expo Secure Store** - تخزين البيانات الحساسة بأمان
- **TypeScript** - للكتابة الآمنة

## البدء

### المتطلبات

- Node.js (الإصدار 16 أو أحدث)
- npm أو yarn
- Expo CLI
- تطبيق Expo Go على الهاتف (للاختبار)

### التثبيت

1. انتقل إلى مجلد التطبيق:
```bash
cd mobile_stockly
```

2. ثبت المتطلبات:
```bash
npm install
```

3. ابدأ خادم التطوير:
```bash
npm start
```

4. امسح رمز QR باستخدام تطبيق Expo Go على هاتفك

### إعداد API

تأكد من تشغيل خادم Django API على:
- التطوير: `http://127.0.0.1:8000`
- الإنتاج: `https://stockly.encryptosystem.com`

يمكنك تعديل عنوان API في ملف `src/config/api.ts`.

## الشاشات المتاحة

### 🏠 لوحة التحكم
- عرض الإحصائيات الأساسية
- الفواتير الأخيرة
- إجراءات سريعة

### 👥 إدارة العملاء
- عرض قائمة العملاء
- البحث والفلترة
- إضافة عملاء جدد

- إضافة منتجات للفاتورة
- تأكيد الفواتير

### 🏷️ إدارة الفئات
- عرض وإدارة فئات المنتجات
- إضافة فئات جديدة
